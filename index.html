ZeroHtml IDE

Mobiweb ZeroHtml IDE est une application Android révolutionnaire qui vise à simplifier le processus de création de sites web dynamiques et statiques en utilisant les langages HTML, CSS et JavaScript. Dotée d'une interface conviviale et de fonctionnalités avancées, ZeroHtml permet aux utilisateurs de concevoir des sites web professionnels en toute simplicité, directement depuis leur appareil mobile. Grâce à des fonctionnalités telles que le glisser-déposer, la génération des pages et des éléments avec l'IA et l'édition de code en temps réel, ZeroHtml offre une expérience de développement web fluide et intuitive.

Drag Drop Interface

En général Drap Drop Interface regroupe l'ensemble des outils pouvant permettre la conception d'une page web et de modifier visuellement le design d'un site web de manière interactive sans écrire aucun code.
En utilisant cette interface, les utilisateurs peuvent simplement sélectionner des éléments prédéfinis tels que des images, des blocs de texte, des formulaires, des boutons, etc., les glisser à l'endroit souhaité sur la page, et les déposer pour les positionner. Cela simplifie le processus de création de sites web pour les personnes qui ne sont pas familières avec le codage HTML et CSS.

Magic Editor

Magic Editor est un outil indispensable pour formater le texte de manière personnalisée dans ZeroHtml. Grâce à cet outil, vous pouvez rédiger des articles et des longs textes en appliquant une variété de styles pour mettre en valeur votre contenu. Voici les différents styles que vous pouvez appliquer avec Magic Editor :

  • Gras (Bold) : Met en évidence du texte en le rendant en gras.
  • Italique (Italic) : Met en valeur du texte en l'inclinant de manière italique.
  • Couleur de fond (Background Color) : Permet de changer la couleur de fond d'un texte ou d'une sélection.
  • Couleur du texte (Text Color) : Permet de modifier la couleur du texte pour le rendre plus attrayant.
  • Taille du texte (Text size) : Permet d'ajuster la taille du texte pour une meilleure lisibilité ou mise en avant.
  • Effaceur (Eraser) : Permet de supprimer le formatage appliqué à un texte.
  • Indentation (Indent) : Permet d'ajouter une indentation au texte pour le structurer.
  • Décalage à gauche (Outdent) : Décale le texte vers la gauche pour le mettre en retrait.
  • Indice (Subscript) : Met en indice certains caractères.
  • Exposant (Superscript) : Met en exposant certains caractères.
  • Barré (Strike through) : Permet de barrer du texte.
  • Titres de section (Heading H1-H6) : Permet de définir des niveaux de titres de section.
  • Paragraphe (Paragraph) : Définit un paragraphe dans le texte.
  • Justifier (Justify) : Justifie le texte.
  • Puces (Bullet) : Crée une liste à puces.
  • Numérotation (Number) : Crée une liste numérotée.
  • Citation (Blockquote) : Met en forme un texte comme une citation.
  • Lien (Link) : Permet d'ajouter des liens hypertextes.
    • Avec ces fonctionnalités de mise en forme, Magic Editor vous offre un large éventail d'options pour personnaliser le style de votre texte dans ZeroHtml, vous permettant ainsi de créer du contenu attractif

    Image Filter

    Image Filter est un simple outil intégré pouvant appliquer des filtres artistiques non seulement à des images, mais aussi à d'autres éléments de la page web comme des buttons, div, text... L'utilisation de filtres artistiques peut ajouter une touche visuelle intéressante et personnalisée à un site web.

    Color Picker - Gradient

    ZeroHtml offre une fonctionnalité de sélecteur de couleur flexible qui s'adapte à vos besoins en matière de design. Vous pouvez utiliser ce sélecteur de couleur pour appliquer des couleurs aux textes, aux éléments de votre site web, définir l'arrière-plan, ainsi que pour insérer des codes de couleurs depuis l'éditeur de code intégré. Ex: Vous n'êtes plus obligé de garder les codes des couleurs, vous pouvez insérer ses couleurs pendant la programmation.

    Meta tags

    Les méta-tags sur une page web jouent un rôle crucial dans le référencement, l'accessibilité et l'expérience utilisateur. Les moteurs de recherche utilisent ces informations pour afficher des résultats pertinents aux utilisateurs. Un titre, une description et keywords bien optimisés peuvent améliorer le classement de votre page dans les résultats de recherche. Les balises méta keywords peuvent être utilisées pour indiquer les mots-clés pertinents liés au contenu de la page.

    Code source - Fetching

    Avec cet outil, vous êtes désormais capable d'afficher le code source de n'importe quel site web, de le copier ou le modifier. Fetching, quand à lui, permet d'extraire les fichiers externe d'un site web comme : css, javascript, Html, des images, video et autres

    Ace Code Editor

    L'éditeur de code Ace est une solution complète et performante intégrée à ZeroHtml qui vous permet d'écrire efficacement et rapidement votre code. Grâce à ses fonctionnalités avancées, Ace Code Editor vous aide à mieux structurer votre code en identifiant les différentes syntaxes grâce à une coloration syntaxique variée et personnalisable à travers différents thèmes. Cela facilite la lecture et la compréhension de votre code, la vérification des erreurs syntaxique, suggestion des codes, ce qui contribue à une expérience de programmation plus fluide et productive.
    En outre, Ace Code Editor prend en charge une large gamme de langages de programmation tels que HTML, CSS, JavaScript, XML, PHP, MySQL, SQL, et bien d'autres. Cela signifie que vous pouvez travailler sur divers types de projets web directement depuis votre appareil mobile en bénéficiant d'un éditeur de code puissant et polyvalent.

    File Tree

    FileTree est un gestionnaire de fichiers arborescent intégré à ZeroHtml qui facilite la gestion des fichiers de votre projet. Il permet de naviguer à travers les fichiers de manière visuelle en les présentant sous forme d'arborescence, avec des icônes spécifiques pour identifier et différencier les différents types de fichiers. Vous pouvez manipuler librement vos fichiers en effectuant des opérations telles que la suppression, le renommage, la lecture des fichiers depuis d'autres applications...

    in-Broswer

    Pour mieux visualiser son travail, un navigateur est intégré pour faciliter l'affichage de nos page web conçu depuis l'interface éditeur (GUI ou Ace Code Editor) avec l'utilisation des outils comme console log et eruda console pour tester et debugger les erreurs, Web2Pdf pour convertir cette page web créée en pdf,...
